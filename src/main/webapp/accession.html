SampleTab Accessioner

    BioSample Database SampleTab Web Accessioner

    This webpage provides a service for assigning accessions to SampleTab format submissions to the BioSample Database.

    Please provide the SampleTab file you wish to accession into the form below. You will then be able to download a version of that file that has had sample and group accessions assigned.

    • Node: This service is currently of beta quality. If you experience any problems, please report them to biosamples@ebi.ac.uk and attach the SampleTab file you were trying to accession.
    • Note: Currently we can only accession SampleTab files under 8MB in size. For files approaching this size the accessioning process may take several minutes.
